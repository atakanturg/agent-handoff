# Agent Handoff Package

This repo is a TypeScript CLI package, not a Cloudflare web deployment.

## Verify

```bash
npm install
npm test
npm run build
```

## Publish

The package name is currently `@atakanturgut/agent-handoff`, and the installed CLI command is `agent-handoff`.

```bash
npm publish --tag beta --access public
```

## Environment variables

None required.
